# Welcome to Disclosurely

## Project info

**URL**: https://disclosurely.com  
**Support**: support@disclosurely.com  

Disclosurely is a secure, GDPR-compliant whistleblowing SaaS platform designed to help organisations receive, manage, and resolve misconduct reports safely and anonymously.

## What technologies are used for this project?

This project is built with:

- Next.js (React framework)  
- TypeScript  
- Supabase (database, auth, storage)  
- Tailwind CSS  
- shadcn-ui
